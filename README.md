# Bookstore
> The Bookstore is a website that allow us to `Display a list of books`, `Add a new book` and `Remove a selected book`.
![2022-03-10_22-48-39](https://user-images.githubusercontent.com/63915024/157726590-0562571e-fcac-468b-8846-1f7a60446bc7.png)

## Setup

### Clone this repository

```bash
$ git clone git@github.com:mukhammadrizooff/Bookstore.git
$ cd Bookstore
```

### Run project

```bash
$ npm install
$ npm run start # this will make webpack watching for your changes in code
```
